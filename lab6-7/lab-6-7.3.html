1. Завдання (Варіант №4):

З об'єкту concerts отримати масив, який буде містити лише імена міст. З масиву потрібно прибрати міста, в яких концерт уже пройшов і відсортувати їх у хронологічному порядку. Результат вивести у консоль.
Очікуваний результат ["Одеса", "Умань", "Харків"]
Застосувати стрілочні функції 

const concerts = {
    Київ: new Date("2020-04-01"),
    Умань: new Date("2025-07-02"),
    Вінниця: new Date("2020-04-21"),
    Одеса: new Date("2025-03-15"),
    Хмельницький: new Date("2020-04-18"),
    Харків: new Date("2025-07-10"),
};

2. Код JavaScript:

const concerts = {
    Київ: new Date("2020-04-01"),
    Умань: new Date("2025-07-02"),
    Вінниця: new Date("2020-04-21"),
    Одеса: new Date("2025-03-15"),
    Хмельницький: new Date("2020-04-18"),
    Харків: new Date("2025-07-10"),
};

const today = new Date();

// Отримати масив міст, де концерти ще не пройшли
const upcomingConcerts = Object.entries(concerts)
    .filter(([city, date]) => date > today)  // Фільтрація концертів, які ще не відбулися
    .sort(([, dateA], [, dateB]) => dateA - dateB)  // Сортування за датами
    .map(([city]) => city);  // Отримання імен міст

console.log(upcomingConcerts);
